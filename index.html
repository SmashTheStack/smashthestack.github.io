Where am I?

This network hosts several different wargames, ranging in difficulty. A wargame, in this context, is an environment that simulates software vulnerabilities and allows for the legal execution of exploitation techniques. A typical scenario would be a system with multiple user accounts. Each user account, or level, introduces a different vulnerable program running with privileges of another user. Exploiting the vulnerability, so that you may gain access to the other account, presents you with a new challenge. Work your way through each level to complete the game and earn bragging rights.
